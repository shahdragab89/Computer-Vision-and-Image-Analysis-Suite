# Computer Vision and Image Analysis Suite

A collection of five desktop applications built from scratch in Python, covering core computer vision and image analysis topics. Each project is self-contained in its own repository.

---

## Projects

### 1. Filtering and Edge Detection
Image preprocessing, noise addition (Gaussian, Uniform, Salt & Pepper), low-pass filtering (Average, Gaussian, Median), edge detection (Sobel, Roberts, Prewitt, Canny), histogram analysis, histogram equalization, frequency-domain filtering, and hybrid image generation.
🔗 [shahdragab89/Filtering-and-edge-detection](https://github.com/shahdragab89/Filtering-and-edge-detection)

### 2. Edge and Boundary Detection (Hough Transform + SNAKE)
Shape detection using Hough Transform for lines, circles, and ellipses. Active Contour Model (SNAKE) using a greedy algorithm for boundary tracing, with chain code representation and perimeter/area computation.
🔗 [shahdragab89/Hough-transform-and-SNAKE](https://github.com/shahdragab89/Hough-transform-and-SNAKE)

### 3. Image Segmentation
Multi-algorithm segmentation including K-Means, Mean Shift, Agglomerative clustering, Otsu thresholding, Spectral thresholding, and Region Growing — all implemented from scratch with a PyQt5 GUI.
🔗 [shahdragab89/Image-Segmentation](https://github.com/shahdragab89/Image-Segmentation)

### 4. Feature Extraction and Matching
Harris corner detector for feature point extraction, SIFT for descriptor generation, and cross-image feature matching using Sum of Squared Differences (SSD) and Normalized Cross-Correlation (NCC), with computation time analysis.
🔗 [shahdragab89/Feature-Extracting-Matching-Tools](https://github.com/shahdragab89/Feature-Extracting-Matching-Tools)

### 5. Face Detection and Recognition
PCA-based face recognition system using eigenspace projection and Euclidean distance classification, with a PyQt5 interface, confusion matrix, and ROC curve evaluation.
🔗 [shahdragab89/Face-Detection-and-Recognition](https://github.com/shahdragab89/Face-Detection-and-Recognition)
